# Knights Travails

A Ruby implementation to find the shortest path for a knight on a chessboard from one position to another.

Example output:

```
> pretty_print_knight_moves([3, 3], [4, 3])
=> Congratulations! You've made it in 3 moves.
The path goes like this:
[3, 3]
[5, 4]
[3, 5]
[4, 3]
```

This repository is built in the context of The Odin Project: https://www.theodinproject.com/lessons/ruby-knights-travails
